  • Project 6: Application Security â€“ Delta Dental

    The Delta Dental Data Breach Project examined how vulnerabilities in third party software applications, like MOVEit, can lead to major cybersecurity incidents impacting millions of individuals. This case study highlighted the critical role of application security in protecting sensitive personal and financial information, especially in industries like healthcare and insurance. This project emphasized how maintaining robust application security is fundamental not only for compliance with standards like HIPPA but also for protecting organizational reputation and customer trust.
